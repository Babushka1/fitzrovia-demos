What you're looking at: a real logistic regression, trained in your browser the moment this page loaded. We generate 4,000 synthetic leases with known ground-truth behaviour, hold out 20% the model never sees, train on the rest by gradient descent, and report accuracy on the held-out set. In production the only change is the input: real lease outcomes from Yardi instead of synthetic ones. View source on this page — the entire model is ~60 lines of commented JavaScript.

How training works: the model starts with all weights at zero and repeats 400 times: predict every lease, compare against what actually happened, nudge each weight in the direction that reduces the total error (gradient descent). The learned column converging to the true column is the proof it works — with real data there is no "true" column; the learned weights are the discovery.

The Entire Model

This is the "AI" — a weighted sum squashed to a probability
// score: weighted sum of resident features
z = b0 + w1·tenure + w2·rentIncrease%
     + w3·tickets90d + w4·slaBreaches
     + w5·sentiment + w6·amenityUses
     + w7·isStudent

// sigmoid: squash score to 0–100%
P(renew) = 1 / (1 + e^(−z))

// training: for 400 epochs
//   error = P(renew) − actualOutcome
//   each weight -= learningRate · error · feature
